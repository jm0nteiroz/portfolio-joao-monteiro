# Portfólio — João Pedro Monteiro Quintas

Portfólio profissional de João Pedro Monteiro Quintas, Product Owner com atuação em produtos digitais B2B, integrações via API, operações e métodos ágeis.

## Conteúdo

- apresentação profissional, experiência e habilidades;
- certificações e ferramentas de gestão;
- download do currículo em PDF;
- demonstrações funcionais dos projetos Nexora, Pulse Hours, Project Flow e BridgeOps;
- dados, nomes e métricas fictícios nas demonstrações.

## Tecnologias

- Next.js e React;
- TypeScript;
- Tailwind CSS;
- Vinext e Vite.

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Validação

```bash
npm test
```

O teste executa o build e confirma a renderização do portfólio, os arquivos das quatro demonstrações e a disponibilidade do currículo.
